#RPi-Nokia5110

Nokia5110 Library for Raspberry Pi based on spidev driver.

##Getting Started
-------

![Wiring Diaglam](RPi-Nokia5110/master/example/Wiring-Diagram.png)

```bash
$ git clone https://github.com/shrhdk/RPi-Nokia5110.git
$ cd RPi-Nokia5110
$ make
$ sudo ./Nokia5110_sample
```
